## Featured Projects (Research)

### 🧬 MetaX — OTF (Operational Taxon-Function) for Metaproteomics
A peptide-centric framework that links peptide sequences to **taxonomy** and **function**, enabling “who is doing what” analysis via **Operational Taxon-Function (OTF)**.

- **Repo:** [byemaxx/MetaX](https://github.com/byemaxx/MetaX)  
- **Docs:** [MetaX Documentation](https://byemaxx.github.io/MetaX/)  
- **Publication:** *Analytical Chemistry* (2025) — “Operational taxon-function framework in MetaX: Unveiling taxonomic and functional associations in metaproteomics”  
  - **DOI:** 10.1021/acs.analchem.4c06645

### 🌳 MetaTree — Browser-based heat-tree visualization for multi-group comparisons
An entirely browser-based platform for interactive visualization of hierarchical quantitative data. MetaTree maps values onto **tree color**, **node size**, and **branch width**, and enables consistent cross-sample and cross-group comparisons by using a **shared reference topology** (all trees share the same structure).

- **Repo:** [byemaxx/MetaTree](https://github.com/byemaxx/MetaTree)  
- **Highlights:**
  - **Shared topology** across all trees to support consistent group comparisons  
  - Accepts hierarchical abundance tables and statistical result tables  
  - Modes for single-sample exploration, group summaries, pairwise comparisons, and all-versus-all comparison matrices  
  - Export figures in raster and vector formats

## Selected Utilities (Bioinformatics)

### 🧬 BlastGUI — cross-platform local BLAST GUI for database building / filtering / alignment
A Python-based cross-platform local BLAST visualization tool that streamlines common BLAST workflows without long command lines.

- **Repo:** [byemaxx/BlastGUI](https://github.com/byemaxx/BlastGUI)  
- **Key features:** local DB building, GUI-based BLAST runs, optional extra parameters, and basic result visualization  
- **Paper:** *Molecular Informatics* (2020) — “BlastGUI: A Python-based Cross-platform Local BLAST Visualization Software.”

### 🧬 SSREnricher — polymorphic SSR enrichment utility for transcript assemblies
A utility to automatically enrich polymorphic SSRs from Trinity transcriptome assemblies.

- **Repo:** [byemaxx/SSREnricher](https://github.com/byemaxx/SSREnricher)  
- **Key features:** GUI + CLI workflows; designed for multi-sample Trinity FASTA inputs  
- **Paper:** *PeerJ* (2020) — “SSREnricher: a computational approach for large-scale identification of polymorphic microsatellites based on comparative transcriptome analysis.”

### 📈 gseaGUI — Enrichment workflow helper (GUI)
A lightweight GUI-style utility to run enrichment workflows more conveniently.

- **Repo:** [byemaxx/gseaGUI](https://github.com/byemaxx/gseaGUI)  
- **Install:** `pip install gseagui`  
- **Includes:** ORA (hypergeometric) + GSEA workflows, basic visualization, and GMT generation

### 🧪 SCFA_Marker — utilities for SCFA-associated signals
Helper scripts for exploring SCFA-associated patterns in multi-omics datasets.

- **Repo:** [byemaxx/SCFA_Marker](https://github.com/byemaxx/SCFA_Marker)

### 🧬 callCodeml — batch runner for PAML codeml branch-site model to scan PSGs
Batch runner for large-scale positive selection scans using PAML codeml branch-site models.

- **Repo:** [byemaxx/callCodeml](https://github.com/byemaxx/callCodeml)
